## v1.2.0 — Помодоро-центричная переработка

### Концепция
Помодоро — центральная механика бота. Рабочий день разбит на автоматические
помодоро-циклы (25+5 мин по умолчанию). Задачи привязываются к дням, а не к
конкретным блокам. В начале каждого помодоро бот спрашивает, какую задачу берёшь.

### 1. Переработка команд бота

**Удалить команды:** /stop, /pause, /resume, /admin

**Оставить:**
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие + кнопка открыть планировщик |
| `/help` | Справка |
| `/settings` | Inline-меню: пауза (15м/30м/1ч/2ч/до конца дня), стоп, возобновить. Для админа — ещё админ-панель |
| `/plan` | План на сегодня: задачи + события. Кнопки: "На завтра", "На эту неделю" |
| `/backlog` | Список задач по статусам с возможностью смены. Фильтры: "На сегодня", "Все", "По категории" |
| `/stats` | Статистика за неделю |

**Правило:** никаких аргументов после команд. Всё через inline-кнопки.

### 2. Помодоро-система

**Автоматические циклы:**
- Первое помодоро = начало рабочего дня (из настроек)
- Цикл: 25 мин работа → 5 мин перерыв → повтор
- Каждый N-й помодоро (по умолчанию 4) → длинный перерыв (по умолчанию 30 мин)
- Во время событий с конкретным временем (созвоны) — помодоро идут фоново, без оповещений

**Настройки помодоро (WebApp → Настройки):**
- Длительность работы (default 25 мин)
- Короткий перерыв (default 5 мин)
- Длинный перерыв (default 30 мин)
- Кол-во помодоро до длинного перерыва (default 4)

**Флоу уведомлений:**
```
[Старт помодоро] → "Какую задачу берёшь?" + список задач на сегодня
[Через 25 мин]   → Опросник: ✅ Выполнено / ⚡ Частично / ❌ Не выполнено
                  → Сразу: "Перерыв 5 мин. Какую задачу возьмёшь дальше?"
[Через 5 мин]    → "Поехали! Помодоро #N"
[Игнор > 2 мин]  → Спам (экспоненциальный, как сейчас)
```

**Итог дня:**
- Кол-во успешных/неуспешных помодоро
- Задачи: выполнено / частично / не выполнено
- Невыполненные задачи → "Перенести на завтра?" (если завтра выходной → на следующий рабочий день)

---

### 3. Переработка задач

**Убрать из задачи:**
- Минимальное время (оставить одно — прогнозируемое)
- Галочка "Помодоро" (теперь всё помодоро)
- Галочка "Разрешить группировку" (нет мульти-задач)
- Типы блоков (fixed/open/range) — убрать

**Статусы задач (вместо старых):**
- `grooming` — в проработке (default)
- `in_progress` — в работе
- `blocked` — заблокирована (не показывать в опроснике помодоро)
- `done` — выполнена (не показывать в опроснике помодоро)

**Смена статуса:** из бота (`/backlog`) и из WebApp

**Привязка к дням:**
- Задача назначается на день (или несколько дней если повторяемая)
- БЕЗ привязки к конкретному времени
- В опроснике помодоро показываются задачи со статусом `grooming` или `in_progress`, назначенные на сегодня

---

### 4. События в календаре (созвоны, встречи)

**Отдельная сущность** — не задача, а событие:
- Название
- Дата + время начала + время окончания (произвольное, не 30-мин слоты)
- Категория (опционально)
- Напоминание за N мин (default 5, настраиваемое для каждого)
- Галочка "Привязать к задаче" → выбор из списка (опционально)

**Создание:** только через календарь в WebApp (клик на слот)
**Уведомления:** напоминание за N мин + уведомление о старте
**Статистика:** отдельно от задач (время на созвоны/встречи)
**Отображение в календаре:** поддержка неровного времени (9:45-10:45)

---

### 5. Календарь — доработки

- Дневной вид на мобильном (переключатель день/неделя)
- Горизонтальный скролл в недельном виде
- Поддержка произвольного времени событий (не только 30-мин слоты)
- Задачи на день — показывать списком сверху/сбоку (без привязки ко времени)
- Фильтр по статусам задач
- Визуальное различение событий и задач

---

### 6. Настройки — реорганизация

**Убрать:** тихое время (quiet_start/quiet_end)

**WebApp настройки:**
- Рабочий день: время начала — время окончания (для каждого дня недели)
- Выходные дни
- Помодоро: длительность работы, короткий перерыв, длинный перерыв, кол-во до длинного
- Часовой пояс
- Время итога дня
- Спам: интервал, множитель, max интервал, вкл/выкл

---

### 8. Баги для исправления

- Часовой пояс: сервер в UTC, бот считает время неправильно. Все расчёты через TZ пользователя
- Спам не работает — проверить и починить
- Дедлайн: окошко с датой съехало в UI
- Если нет задач по блокам — НЕ оповещать (убрать "спам для пустых слотов")

## Выполнено (v1.0.0 — v1.1.0)

### Деплой (2026-03-29)
- Backend на VPS через systemd + NPM (Nginx Proxy Manager)
- SSL через Let's Encrypt (NPM)
- Frontend на Cloudflare Pages (auto-deploy из GitHub)
- Telegram webhook настроен

### Система доступа
- Whitelist middleware (aiogram + FastAPI)
- Запрос доступа для новых пользователей
- Admin-панель для управления пользователями

### Выбор часового пояса
- Автоопределение через Intl API
- 24 строки с городами, DST-индикатор
- Доступен при первом запуске и в настройках

### Основной функционал
- Категории (CRUD, emoji, цвет, drag-сортировка)
- Бэклог задач (приоритеты, дедлайны, теги)
- Недельный календарь (drag & drop блоков)
- Расписание недели
- Цели по категориям
- Напоминания через Telegram
- Опросник завершения блока + спам при игноре
- Pomodoro-таймер
- Статистика и саммари
